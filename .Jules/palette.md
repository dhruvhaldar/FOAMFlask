## 2025-05-15 - Grouped Tutorial Dropdown
**Learning:** Organizing long dropdown lists with `<optgroup>` significantly improves scannability and reduces cognitive load, especially for hierarchical data like file paths. Grouping by top-level directory clarifies context without cluttering the display text.
**Action:** When presenting hierarchical options (like file paths or categorized items), parse the structure and use `<optgroup>` to group related items, stripping redundant prefixes from the display text for cleaner UI.
